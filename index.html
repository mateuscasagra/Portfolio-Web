Projeto Relevante

Bot Meta

Bot Meta é uma aplicação desenvolvida com o foco no atendimento automático para fast foods utilizando a tecnologia da Meta Cloud-Api.

As tecnologias utilizadas são JavaScript, Django, Python, Cloud API, PostgreSQL, AWS, Gunicorn e Nginx

Tudo começou quando comecei a estudar como automatizar mensagens pelo WhatsApp utilizando a biblioteca WPPConnect. Após três meses de estudo, decidi migrar para a API oficial da Meta, a Cloud API. O bot foi desenvolvido pensando em uma barraca de cachorro-quente famosa em meu bairro, que recebia muitos pedidos por WhatsApp e tinha um tempo de espera de 10 minutos para atendimento.
